# 华彩系统一期接口清单

## 1. 文档说明

本清单基于当前原型和《技术方案、数据库实体清单与一期开发范围》整理，目标是给前后端联调和后端接口开发提供一份可以直接落地的接口边界说明。

适用范围：

- 一期 MVP
- Vue3 管理后台
- Spring Boot 3 后端
- MySQL 8 数据模型


## 2. 接口设计约定

### 2.1 基础约定

- Base URL：`/api/v1`
- 认证方式：`Bearer Token`
- 数据格式：`application/json`
- 文件上传：`multipart/form-data`
- 时间格式：`yyyy-MM-dd HH:mm:ss`

### 2.2 通用响应结构

```json
{
  "code": 0,
  "message": "success",
  "data": {},
  "traceId": "e7e2b0c4b4d24d1e"
}
```

### 2.3 分页响应结构

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "records": [],
    "total": 100,
    "pageNum": 1,
    "pageSize": 20
  }
}
```

### 2.4 通用分页请求参数

- `pageNum`
- `pageSize`
- `keyword`
- `sortField`
- `sortOrder`

### 2.5 状态字段建议

- `status`
- `auditStatus`
- `accountStatus`
- `employmentStatus`
- `capitalSourceType`

建议统一使用字典枚举值，不在前端硬编码。


## 3. 认证与登录

### 3.1 登录

- `POST /api/v1/auth/login`
- 说明：账号密码登录

请求体：

```json
{
  "username": "admin",
  "password": "123456"
}
```

响应体：

```json
{
  "token": "jwt-token",
  "tokenType": "Bearer",
  "expiresIn": 7200,
  "userInfo": {
    "id": 1,
    "username": "admin",
    "realName": "管理员"
  }
}
```

### 3.2 获取当前登录人信息

- `GET /api/v1/auth/me`

### 3.3 修改密码

- `PUT /api/v1/auth/password`

请求体：

```json
{
  "oldPassword": "123456",
  "newPassword": "654321"
}
```

### 3.4 退出登录

- `POST /api/v1/auth/logout`

## 11. 当前仍需确认的接口口径

- 客户是否允许删除，还是只允许停用
- 客户状态变更是手工维护还是由借贷/还款自动驱动
- 借贷余额是否实时计算，还是录入时直接存储
- 还款本金与利息是否允许手工录入后校验
- 收入/支出是否要和客户、借贷单据联动
- 导入失败明细是否需要单独落表
